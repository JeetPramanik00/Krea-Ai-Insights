# Krea Pricing Page — Agent Implementation Instructions

## Tech Stack
- **React + Vite + TypeScript**
- **Tailwind CSS** (utility-first, no component libraries)
- **No external UI kits** — everything hand-built

---

## 1. Global Design Tokens

```
Background: #0a0a0a (near-black)
Surface cards: #111111 / #141414
Border color: #222222 (subtle)
Primary accent: #4f6ef7 (blue — used on Pro badge, CTA buttons)
Green accent: #22c55e (checkmarks, "Most Popular" indicators)
Text primary: #ffffff
Text secondary: #888888 / #666666
Text muted: #444444
Font family: Inter (Google Fonts) — weights 300, 400, 500, 600, 700
Border radius on cards: 12px–16px
```

## 4. Component Notes

- **Monthly/Yearly Toggle**: Use a pill toggle with smooth transition. Toggling should update all prices.
- **"New" badges**: Small inline pills, blue background, white text, `text-xs`, `rounded-full`, `px-2 py-0.5`
- **"Coming Soon" badge**: Same pill style but different context (FAQ section)
- **Checkmarks**: Use a custom SVG circle-check icon in `#22c55e`
- **CTA Buttons**:
  - Primary (Pro): `bg-[#4f6ef7] text-white hover:bg-blue-600`
  - Outlined: `border border-[#333] text-white hover:border-white`
- **Cards**: `bg-[#111] border border-[#1f1f1f] rounded-2xl p-6`
- **Compute chip illustration**: Build as SVG or use a grid-based CSS art — dark rectangle, corner LED dots in green, grid lines inside

## Final Agent Notes

- Do **not** use any UI component libraries (no shadcn, no Radix, no MUI)
- All dark surfaces must be slightly different shades — avoid using a single flat black
- The Pro card must visually stand out from the other two — use a blue top border `2px solid #4f6ef7` or a faint blue glow `box-shadow`
- Spacing discipline: use Tailwind's spacing scale consistently (`gap-6`, `py-16`, `px-8`)
- The compute pack chip graphic is central to Section 4 — invest time in making it look like a stylized GPU/chip illustration, not a placeholder box
